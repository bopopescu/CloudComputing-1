registerTempTable {SparkR}R Documentation

Register Temporary Table

Description

Registers a DataFrame as a Temporary Table in the SQLContext

Usage

## S4 method for signature 'DataFrame,character'
registerTempTable(x, tableName)

registerTempTable(x, tableName)

Arguments

x

A SparkSQL DataFrame

tableName

A character vector containing the name of the table

Examples

## Not run: 
sc <- sparkR.init()
sqlContext <- sparkRSQL.init(sc)
path <- "path/to/file.json"
df <- jsonFile(sqlContext, path)
registerTempTable(df, "json_df")
new_df <- sql(sqlContext, "SELECT * FROM json_df")

## End(Not run)
